What is the difference between Java SE and Java ME? 
Answer: 
Java SE
-Used for developing desktop applications,enterprise applications, and general-purpose software.
-Runs on desktops,laptops and servers.
-Requires more memory and processing power.
-JavaFX, Swing, JDBC, Servlets etc.

Java ME
-Designed for mobile devices,embedded systems, and IoT applications. 
-Runs on mobile phones,smart cards, and embedded devices. 
-Optimized for low memory and limited processing power. 
-MIDlets(Mobile Information Device Applications), CLDC(Connected Limited Device Configuration).
